Pour tester en local :

1. Installe les dépendances Node.js nécessaires :
   npm install express
   npm install @babylonjs/core @babylonjs/loaders

2. Lance le serveur :
   node server.js

3. Ouvre http://localhost:3000 dans ton navigateur.

Le code Babylon.js est dans public/main.js (modulaire, prêt à être adapté à React plus tard).
